# นโยบายความเป็นส่วนตัว 2syn (Privacy Policy)

อัปเดตล่าสุด: 26 พฤษภาคม 2026

2syn ใช้การเข้ารหัสแบบ End-to-End (E2EE) เราไม่เคยดักจับ จัดเก็บ หรือวิเคราะห์หน้าจอ เสียง หรือข้อมูลอินพุตของคุณ

## ข้อมูลที่เก็บรวบรวม
1. Hardware ID (HWID): สำหรับการผูกใบอนุญาต
2. ข้อมูลการส่งสัญญาณ (SDP/ICE): สำหรับการเชื่อมต่อ P2P
