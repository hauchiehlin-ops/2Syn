# คู่มือผู้ใช้ 2syn

ยินดีต้อนรับสู่ **2syn**! 2syn คือระบบเดสก์ท็อประยะไกลประสิทธิภาพสูงและปลอดภัย ที่ผสมผสานการเข้ารหัส WebRTC แบบ end-to-end กับเทคโนโลยีการปรับอัตราบิตแบบอัตโนมัติ รองรับการเชื่อมต่อข้ามแพลตฟอร์มจากโฮสต์ macOS/Windows ไปยังไคลเอนต์ iOS/macOS

---

## 1. การจับคู่การเชื่อมต่อ

### 1.1 รับ ID โฮสต์
เปิด 2syn บน **โฮสต์** (คอมพิวเตอร์ที่จะถูกควบคุม):
- ID 9 หลักจะปรากฏในช่อง「My ID」ที่ด้านบนของหน้าจอ (เช่น `569-639-684`)
- คลิกปุ่ม 📋 ข้าง ID เพื่อคัดลอก หรือคลิกปุ่ม ⬛ เพื่อแสดง QR code

### 1.2 เชื่อมต่อจากอุปกรณ์ iOS
1. เปิดแอป 2syn บน iOS
2. ป้อน ID โฮสต์ด้วยตนเองในช่อง「Connect to」หรือให้โฮสต์แสดง QR code แล้วสแกนด้วยกล้องดั้งเดิมของ iOS — ID จะถูกกรอกโดยอัตโนมัติ
3. แตะ「Connect」ระบบจะทำ NAT traversal ผ่าน STUN โดยอัตโนมัติและสร้างการเชื่อมต่อโดยตรงแบบ end-to-end — ไม่ต้องตั้งค่า Port Forwarding บนเราเตอร์

---

## 2. การควบคุมระยะไกล

### 2.1 การสลับโหมดควบคุม
หลังเชื่อมต่อแล้ว แตะปุ่ม ⚙️ บนแถบเครื่องมือที่มุมขวาบนเพื่อขยายแผงและสลับระหว่างโหมดควบคุม 2 โหมด:

| โหมด | คำอธิบาย |
|---|---|
| **Trackpad（โหมดทัชแพด）** | ลากนิ้ว = เลื่อนเคอร์เซอร์；แตะนิ้วเดียว = คลิกซ้าย；เลื่อนสองนิ้วขึ้นลง = เลื่อน；แตะสองนิ้ว = คลิกขวา |
| **Direct Touch（โหมดสัมผัสโดยตรง）** | พิกัดการสัมผัสตรงกับพิกัดหน้าจอโฮสต์โดยตรง |

### 2.2 การเลื่อนด้วยสองนิ้ว
ในโหมด Trackpad ให้เลื่อนสองนิ้วขึ้นหรือลงบนหน้าจอเพื่อส่งเหตุการณ์เลื่อนไปยังโฮสต์ ใช้งานได้กับทุกแอปพลิเคชัน

### 2.3 การพิมพ์แป้นพิมพ์
1. แตะไอคอนแป้นพิมพ์บนแถบเครื่องมือเพื่อเรียกแป้นพิมพ์บนหน้าจอ iOS
2. แถบเครื่องมือปุ่มตัวแก้ไข (Esc, Tab, ⌃, ⌥, ⌘, ⇧, ปุ่มลูกศร) จะปรากฏเหนือแป้นพิมพ์เพื่อส่งคีย์ลัด
3. แตะปุ่มตัวแก้ไขหนึ่งครั้งเพื่อล็อก (สีน้ำเงินสว่าง) จะปลดล็อกโดยอัตโนมัติหลังกดปุ่มหนึ่งปุ่ม แตะอีกครั้งเพื่อปลดล็อกด้วยตนเอง

### 2.4 การตรวจจับแรงกด Apple Pencil
บน iPad ที่รองรับ Apple Pencil เขียนบนหน้าจอโดยตรงด้วย Pencil 2syn จะส่ง:
- ค่าแรงกด (0–100%)
- มุมเอียงของปากกา (แกน X/Y ±90°)

ไปยังโฮสต์แบบเรียลไทม์ แอปพลิเคชันระดับมืออาชีพเช่น Procreate และ Adobe Photoshop จะได้รับข้อมูลแรงกดและการเอียงเต็มรูปแบบ

---

## 3. การสตรีมเสียง

เสียงระบบของโฮสต์ (รวมถึงเสียงแอป เพลง ฯลฯ) จะถูกสตรีมไปยังอุปกรณ์ iOS ของคุณโดยอัตโนมัติ

- **ไม่มีเสียงบน iOS**: แตะปุ่ม「🔇 แตะเพื่อเปิดเสียง」ที่มุมขวาบน (นโยบายความปลอดภัยของเบราว์เซอร์ iOS กำหนดให้ผู้ใช้ทำท่าทางก่อนเล่นเสียง)
- หลังเชื่อมต่อแล้ว แตะ「🔊 Mute」เพื่อสลับปิดเสียง

---

## 4. การซิงค์คลิปบอร์ด

- **โฮสต์ → iOS**: หลังคัดลอกข้อความบนโฮสต์ การแจ้งเตือน Toast จะปรากฏที่ด้านล่างของหน้าจอ iOS พร้อมแสดงตัวอย่างเนื้อหาที่คัดลอก แตะ Toast เพื่อเขียนข้อความไปยังคลิปบอร์ดในเครื่องของ iOS
- **iOS → โฮสต์**: พิมพ์บนแป้นพิมพ์ iOS จากนั้นกดค้างเพื่อวาง — ข้อมูลที่ป้อนจะถูกส่งไปยังช่องที่โฟกัสบนโฮสต์โดยตรง

---

## 5. ขนาดการแสดงผล

ปุ่ม「🔍 ขนาดดั้งเดิม / พอดีหน้าจอ」บนแถบเครื่องมือสลับระหว่างโหมดการแสดงผล 2 โหมด:

| โหมด | คำอธิบาย |
|---|---|
| **พอดีหน้าจอ** | ปรับขนาดหน้าจอโฮสต์เพื่อเติมจอแสดงผล iOS ทั้งหมด |
| **ขนาดดั้งเดิม** | การแสดงผลแบบ 1:1 พิกเซล พร้อมรองรับการเลื่อน เหมาะสำหรับงานที่ต้องการความแม่นยำ |

---

## 6. อัตราบิตแบบปรับตัว（ABR）

2syn มีการปรับคุณภาพอัตโนมัติในตัว ตรวจจับ RTT ของเครือข่ายและการสูญเสียแพ็กเก็ตทุก 500 มิลลิวินาที:

| สภาพเครือข่าย | การปรับอัตโนมัติ |
|---|---|
| ดี（RTT < 80 มิลลิวินาที, การสูญเสีย < 1%）| คุณภาพสูง อัตราเฟรมสูง |
| ปานกลาง | คุณภาพปานกลาง |
| แย่（RTT > 200 มิลลิวินาที หรือการสูญเสีย > 5%）| ลดอัตราเฟรมและอัตราบิตเพื่อรักษาเสถียรภาพการเชื่อมต่อ |

ตัวบ่งชี้จุดที่มุมขวาบน (เขียว / เหลือง / แดง) สะท้อนคุณภาพการเชื่อมต่อแบบเรียลไทม์

---

## 7. โหมดหน้าจอดำเพื่อความเป็นส่วนตัว

ทำเครื่องหมายที่「โหมดความเป็นส่วนตัว」ในอินเตอร์เฟซ 2syn บน **โฮสต์** เพื่อทำให้หน้าจอโฮสต์ดำลง ป้องกันไม่ให้ผู้สังเกตการณ์เห็นการดำเนินการของคุณ การควบคุมระยะไกลยังคงทำงานได้ตามปกติ

---

## 8. การถ่ายโอนไฟล์

เมื่อการเชื่อมต่อเสถียร (การเชื่อมต่อโดยตรง P2P) ให้ลากไฟล์ไปยังหน้าต่าง 2syn จากฝั่ง iOS หรือฝั่งโฮสต์เพื่อเริ่มการถ่ายโอน การถ่ายโอนทั้งหมดได้รับการเข้ารหัสแบบ end-to-end ผ่าน WebRTC Data Channel

---

## 9. การตัดการเชื่อมต่อและการเชื่อมต่อใหม่

แตะ ⚙️ → 「🚪 ออกจากระบบ」บนแถบเครื่องมือเพื่อสิ้นสุดเซสชัน หากการเชื่อมต่อหลุดโดยไม่คาดคิด แอปจะแสดงข้อความแจ้ง — ป้อน ID อีกครั้งเพื่อเชื่อมต่อใหม่

---

## 10. คำถามที่พบบ่อย

**ถาม: การเชื่อมต่อค้างอยู่ที่ "กำลังเชื่อมต่อ..." และไม่สามารถสร้างได้?**
ตอบ: ทั้งสองฝ่ายอยู่หลังไฟร์วอลล์องค์กรที่เข้มงวด (Symmetric NAT) อาจทำให้ NAT traversal ล้มเหลว ลองเปลี่ยนฝ่ายหนึ่งไปใช้เครือข่ายมือถือ 4G/5G แล้วลองใหม่

**ถาม: ไม่มีเสียงบน iOS?**
ตอบ: แตะปุ่ม「🔇 แตะเพื่อเปิดเสียง」บนหน้าจอ iOS ต้องการท่าทางของผู้ใช้เพื่อปลดล็อกการเล่นเสียง

**ถาม: วิดีโอไม่ชัดหรือกระตุก?**
ตอบ: อัตราบิตแบบปรับตัว (ABR) จะปรับคุณภาพโดยอัตโนมัติตามสภาพเครือข่าย คุณภาพจะลดลงเพื่อรักษาความลื่นไหลบนเครือข่ายที่แย่ และฟื้นตัวโดยอัตโนมัติเมื่อเครือข่ายดีขึ้น

**ถาม: Apple Pencil ไม่มีเอฟเฟกต์แรงกด?**
ตอบ: ตรวจสอบให้แน่ใจว่า 2syn บน iPad ของคุณเป็นเวอร์ชันล่าสุด และ 2syn บนโฮสต์ได้รับการอัปเดตเป็น v3.5.11 หรือใหม่กว่า

**ถาม: ไม่สามารถเชื่อมต่อได้หลังสแกน QR code?**
ตอบ: QR code มีเพียง ID เท่านั้น ยืนยันว่า 2syn บนโฮสต์กำลังทำงานและแสดง ID เดียวกัน
