# 2syn 用户操作手册 (User Manual)

欢迎使用 2syn！2syn 是一款结合跨平台原生高性能与 WebRTC 端到端加密的安全远程控制系统。

## 1. 快速开始与设备授权
1. 启动 2syn，在输入框中填写密钥（测试阶段：`BUYOUT-KEY-12345`）。
2. 双方设备授权后，在“我要连接”栏输入受控端识别码。
3. 自动 NAT 穿透：系统将自动通过 STUN 建立直连通道。

## 2. 界面操作与性能调整
* **智能自动质量优化**：每 500ms 侦测延迟。流畅时启用 144Hz 与 YUV444；拥堵时降级至 60/30 FPS 与 YUV420。
* **收合式高级控制面板**：手动强制色彩格式、限制最高帧率（FPS）与比特率（Bitrate）。
* **文件传输**：支持跨设备双向拖拽传输，全程端到端加密。

## 3. 隐私黑屏模式
远程办公时，点击“隐私黑屏”，系统会插入虚拟显示器并关闭物理屏幕，防止旁人偷窥。

## 4. 常见问题排查
* **一直卡在“正在交握 SDP/ICE”**：网络为 Symmetric NAT，打洞失败。请确保一方使用 4G/5G 或一般家用 Wi-Fi，或配置 TURN 服务器。
* **设备数达上限 (err_limit_exceeded)**：前往旧设备点击“解除绑定”。
* **画面模糊**：网络不稳导致自动降级为 YUV420，可在设置中强制恢复 YUV444。
