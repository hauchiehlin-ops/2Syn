# Руководство пользователя 2syn

Добро пожаловать в **2syn**! 2syn — это высокопроизводительная и безопасная система удалённого рабочего стола, сочетающая сквозное шифрование WebRTC и технологию адаптивного битрейта, поддерживающая кроссплатформенные подключения от хостов macOS/Windows к клиентам iOS/macOS.

---

## 1. Подключение и сопряжение

### 1.1 Получение ID хоста
Запустите 2syn на **хосте** (управляемом компьютере):
- В поле「My ID」в верхней части экрана отобразится 9-значный идентификатор (например, `569-639-684`).
- Нажмите кнопку 📋 рядом с ID, чтобы скопировать его, или кнопку ⬛ для отображения QR-кода.

### 1.2 Подключение с iOS-устройства
1. Запустите приложение 2syn на iOS.
2. Вручную введите ID хоста в поле「Connect to」или попросите хост показать QR-код и отсканируйте его встроенной камерой iOS — ID заполнится автоматически.
3. Нажмите「Connect」. Система автоматически выполнит обход NAT через STUN и установит прямое сквозное соединение — настройка проброса портов на роутере не требуется.

---

## 2. Удалённое управление

### 2.1 Переключение режимов управления
После подключения нажмите кнопку ⚙️ на панели инструментов в правом верхнем углу, чтобы развернуть панель и переключаться между двумя режимами управления:

| Режим | Описание |
|---|---|
| **Trackpad（Режим трекпада）** | Перетаскивание пальцем = перемещение курсора；одиночное касание = левая кнопка мыши；прокрутка двумя пальцами = прокрутка；двойное касание двумя пальцами = правая кнопка мыши |
| **Direct Touch（Режим прямого касания）** | Координаты касания напрямую соответствуют координатам экрана хоста |

### 2.2 Прокрутка двумя пальцами
В режиме Trackpad проведите двумя пальцами вверх или вниз по экрану, чтобы отправить события прокрутки хосту. Работает со всеми приложениями.

### 2.3 Ввод с клавиатуры
1. Нажмите значок клавиатуры на панели инструментов для вызова экранной клавиатуры iOS.
2. Над клавиатурой появится панель клавиш-модификаторов (Esc, Tab, ⌃, ⌥, ⌘, ⇧, клавиши со стрелками) для отправки комбинаций быстрых клавиш.
3. Нажмите клавишу-модификатор один раз для блокировки (подсвечивается синим); она автоматически разблокируется после нажатия одной клавиши. Нажмите снова для ручной разблокировки.

### 2.4 Чувствительность к давлению Apple Pencil
На iPad с поддержкой Apple Pencil пишите прямо на экране. 2syn передаёт:
- Значение давления (0–100%)
- Угол наклона пера (ось X/Y ±90°)

на хост в реальном времени. Профессиональные приложения, такие как Procreate и Adobe Photoshop, получат полные данные о давлении и наклоне.

## 3. Потоковая передача звука

Системный звук хоста (включая звуки приложений, музыку и т.д.) автоматически передаётся на ваше устройство iOS.

- **Нет звука на iOS**: Нажмите кнопку「🔇 Нажмите для включения звука」в правом верхнем углу (политика безопасности браузера iOS требует жеста пользователя перед воспроизведением звука).
- После подключения нажмите「🔊 Mute」для переключения отключения звука.

---

## 4. Синхронизация буфера обмена

- **Хост → iOS**: После копирования текста на хосте в нижней части экрана iOS появится уведомление Toast с предпросмотром скопированного содержимого. Нажмите Toast, чтобы записать текст в локальный буфер обмена iOS.
- **iOS → Хост**: Введите текст на клавиатуре iOS, затем удерживайте для вставки — ввод отправляется напрямую в сфокусированное поле на хосте.

---

## 5. Размер отображения

Кнопка「🔍 Исходный размер / По размеру экрана」на панели инструментов переключает между двумя режимами отображения:

| Режим | Описание |
|---|---|
| **По размеру экрана** | Экран хоста масштабируется для заполнения всего дисплея iOS |
| **Исходный размер** | Отображение пиксель в пиксель с поддержкой панорамирования, идеально для точной работы |

---

## 6. Адаптивный битрейт（ABR）

В 2syn встроена автоматическая регулировка качества: каждые 500 мс обнаруживаются RTT сети и потеря пакетов:

| Состояние сети | Автоматическая регулировка |
|---|---|
| Хорошее（RTT < 80 мс, потери < 1%）| Высокое качество, высокая частота кадров |
| Среднее | Среднее качество |
| Плохое（RTT > 200 мс или потери > 5%）| Снижение частоты кадров и битрейта для поддержания стабильности соединения |

Индикатор точки в правом верхнем углу (зелёный / жёлтый / красный) отражает качество соединения в реальном времени.

---

## 7. Режим конфиденциальности (чёрный экран)

Установите флажок「Режим конфиденциальности」в интерфейсе 2syn на **хосте**, чтобы затемнить экран хоста и предотвратить просмотр ваших операций посторонними. Удалённое управление продолжает работать в обычном режиме.

---

## 8. Передача файлов

При стабильном соединении (прямое P2P-соединение) перетащите файлы в окно 2syn с любой стороны (iOS или хост) для запуска передачи. Все передачи шифруются сквозным образом через WebRTC Data Channel.

---

## 9. Отключение и переподключение

Нажмите ⚙️ → 「🚪 Выйти」на панели инструментов для завершения сеанса. Если соединение неожиданно разрывается, приложение покажет подсказку — введите ID повторно для переподключения.

---

## 10. Часто задаваемые вопросы

**В: Соединение зависло на «Подключение...» и не устанавливается?**
О: Нахождение обеих сторон за строгими корпоративными брандмауэрами (Symmetric NAT) может привести к сбою обхода NAT. Попробуйте переключить одну из сторон на мобильную сеть 4G/5G и повторите попытку.

**В: Нет звука на iOS?**
О: Нажмите кнопку「🔇 Нажмите для включения звука」на экране. iOS требует жест пользователя для разблокировки воспроизведения звука.

**В: Размытое или заторможенное видео?**
О: Адаптивный битрейт (ABR) автоматически регулирует качество в зависимости от состояния сети. Качество снижается для поддержания плавности при плохой сети и автоматически восстанавливается при улучшении сети.

**В: Apple Pencil не реагирует на давление?**
О: Убедитесь, что 2syn на вашем iPad является последней версией, а 2syn на хосте обновлён до v3.5.11 или выше.

**В: Не удаётся подключиться после сканирования QR-кода?**
О: QR-код содержит только ID. Убедитесь, что 2syn на хосте запущен и отображает тот же ID.
