# 2syn Política de Privacidad (Privacy Policy)

Última actualización: 26 de mayo de 2026

2syn utiliza cifrado de extremo a extremo (E2EE). NUNCA interceptamos, almacenamos ni analizamos sus pantallas, audio o datos de entrada.

## Información recopilada
1. ID de hardware (HWID): Para la vinculación de licencias.
2. Datos de señalización (SDP/ICE): Solo para conexión P2P.
