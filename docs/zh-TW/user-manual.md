# 2syn 使用者操作手冊 (User Manual)

歡迎使用 2syn！2syn 是一款結合跨平台原生高效能與 WebRTC 端對端加密的安全遠端桌面控制系統。本手冊將引導您完成初始設定、裝置配對與進階效能調整。

## 1. 快速開始與裝置授權

### 1.1 取得與輸入授權金鑰
2syn 採用一次性買斷授權機制。當您首次啟動應用程式時，畫面將提示您輸入授權金鑰（License Key）。
1. 啟動 2syn 應用程式（無論是桌面版或行動版）。
2. 在輸入框中填寫您的金鑰（測試階段請使用：`BUYOUT-KEY-12345`）。
3. 系統將自動捕捉您本機的硬體特徵碼（HWID）並向信令伺服器完成綁定。每個金鑰最多支援同時授權 5 台設備。

### 1.2 跨裝置連線配對
為確保安全，2syn 使用去中心化的 P2P 架構。當兩台設備皆通過授權驗證後：
1. **主控端與受控端**：在「我要連線」的欄位輸入受控端的識別碼。
2. **自動 NAT 穿透**：系統將自動透過 STUN 伺服器探測雙方的公網 IP，建立端對端的直連通道。您無需額外設定路由器的 Port Forwarding。

## 2. 介面操作與效能調整

### 2.1 智慧自動品質最佳化 (Smart Optimization)
2syn 內建智慧監控模組。系統會以 500 毫秒為週期，自動偵測當前的網路延遲（RTT）與封包遺失率（Packet Loss）：
* **網路順暢時（延遲 < 30ms）**：自動切換至最高畫質（YUV444 無損級色彩）與 144 FPS 電競級更新率。
* **網路壅塞時**：系統會動態調降更新率（至 60 或 30 FPS）並改用 YUV420 壓縮格式，確保連線不中斷且操作不卡頓。

### 2.2 收合式進階控制面板
為保持介面簡潔，影響操作體驗的進階參數預設隱藏在「收合面板」中。點擊介面上的齒輪或「進階設定」按鈕展開：
* **色彩格式**：手動強制選擇 YUV444（高畫質）或 YUV420（省頻寬）。
* **最高更新率 (FPS)**：限制畫面的傳輸上限以節省電量。
* **最高位元率 (Bitrate)**：在行動網路下，可限制最高傳輸頻寬（如 5000 kbps）以避免流量超額。

### 2.3 檔案傳輸功能
當連線品質穩定（非中繼模式且封包遺失率低）時，介面會解鎖檔案傳輸按鈕。您可以直接在兩台設備間拖曳或點選傳輸檔案。所有傳輸皆經過 WebRTC 資料通道（Data Channel）進行端對端加密。

## 3. 安全與隱私保護功能

### 3.1 隱私黑屏模式 (Privacy Black Screen)
當您在遠端操作公司或家裡的電腦時，若不希望螢幕顯示您的操作畫面：
1. 點擊控制列上的「隱私黑屏」按鈕。
2. 系統會自動透過虛擬顯示器驅動（IDD）插入一個全虛擬的 1080p 螢幕，同時關閉實體螢幕的畫面輸出。

## 4. 常見問題與障礙排除 (Troubleshooting)

**Q：連線一直卡在「正在交握 SDP/ICE...」怎麼辦？**
A：這通常代表雙方處於嚴格的防火牆內（Symmetric NAT），導致 P2P 打洞失敗。
* **解決方案**：2syn 已支援 TURN 伺服器中繼備援。在產品化的最終版本中，系統將會自動降級至中繼模式。開發階段請確保其中一方使用較為開放的網路（如 4G/5G 手機網路或一般家用 Wi-Fi）。

**Q：出現「達到最高授權設備數量上限 (err_limit_exceeded)」？**
A：您的金鑰已綁定滿 5 台設備。
* **解決方案**：請至不再使用的設備上點選「解除綁定」。注意：解除綁定有安全冷卻時間設計，避免金鑰遭到濫用。

**Q：畫面感覺有點模糊或顏色黯淡？**
A：系統可能偵測到網路不穩，自動啟用了 `YUV420` 降級模式。若您確認網路頻寬充足，可進入進階設定中強制切換回 `YUV444`。
