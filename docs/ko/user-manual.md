# 2syn 사용자 매뉴얼

**2syn**에 오신 것을 환영합니다! 2syn은 WebRTC 엔드투엔드 암호화와 적응형 비트레이트 기술을 결합한 고성능 보안 원격 데스크톱 시스템으로, macOS/Windows 호스트와 iOS/macOS 클라이언트 간 크로스 플랫폼 연결을 지원합니다.

---

## 1. 연결 페어링

### 1.1 호스트 ID 가져오기
**호스트** 단말(제어받는 PC)에서 2syn을 실행합니다:
- 화면 상단 「My ID」 필드에 9자리 식별 코드가 표시됩니다（예: `569-639-684`）.
- ID 옆의 📋 버튼을 클릭하여 복사하거나, ⬛ 버튼을 클릭하여 QR 코드를 표시할 수 있습니다.

### 1.2 iOS 기기에서 연결하기
1. iOS에서 2syn 앱을 실행합니다.
2. 「Connect to」 필드에 호스트 ID를 직접 입력하거나, 상대방에게 QR 코드를 보여달라고 하여 iOS 기본 카메라로 스캔하면 자동으로 입력됩니다.
3. 「Connect」를 탭합니다. 시스템이 STUN을 통해 자동으로 NAT 통과를 수행하고 엔드투엔드 직접 연결을 설정합니다. 라우터 Port Forwarding 설정이 필요 없습니다.

---

## 2. 원격 제어

### 2.1 제어 모드 전환
연결 후 오른쪽 상단의 ⚙️ 툴바 버튼을 탭하여 패널을 펼치고 두 가지 제어 모드를 전환할 수 있습니다:

| 모드 | 설명 |
|---|---|
| **Trackpad（트랙패드 모드）** | 손가락 드래그 = 커서 이동；단일 탭 = 왼쪽 클릭；두 손가락 위아래 스와이프 = 스크롤；두 손가락 탭 = 오른쪽 클릭 |
| **Direct Touch（직접 터치 모드）** | 터치 좌표가 호스트 화면 좌표에 직접 대응 |

### 2.2 두 손가락 스크롤
Trackpad 모드에서 두 손가락으로 화면을 위아래로 스와이프하면 호스트에 스크롤 이벤트가 전송됩니다. 모든 애플리케이션을 지원합니다.

### 2.3 키보드 입력
1. 툴바의 키보드 아이콘을 탭하여 iOS 화면 키보드를 불러옵니다.
2. 키보드 위에 수정자 키 툴바(Esc, Tab, ⌃, ⌥, ⌘, ⇧, 방향키)가 나타나 단축키 조합을 전송할 수 있습니다.
3. 수정자 키를 한 번 탭하면 잠금(파란색으로 밝아짐)되고, 키를 하나 전송하면 자동 해제됩니다. 다시 탭하면 수동 해제됩니다.

### 2.4 Apple Pencil 필압 감지
Apple Pencil을 지원하는 iPad에서 Pencil로 화면에 직접 쓰면 2syn이 다음을 전송합니다:
- 필압 값（0–100%）
- 펜 기울기 각도（X/Y 축 ±90°）

호스트에 실시간으로 전송되며 Procreate, Adobe Photoshop 등의 전문 앱에서 완전한 필압 및 기울기 정보를 활용할 수 있습니다.

---

## 3. 오디오 스트리밍

호스트의 시스템 오디오(앱 소리, 음악 등)가 자동으로 iOS 기기에 스트리밍됩니다.

- **iOS에서 소리가 없는 경우**: 오른쪽 상단의 「🔇 탭하여 오디오 활성화」 버튼을 탭하세요（iOS 브라우저 보안 정책에 따라 첫 번째 재생은 사용자 제스처가 필요합니다）.
- 연결 후 「🔊 Mute」를 탭하여 음소거를 전환할 수 있습니다.

---

## 4. 클립보드 동기화

- **호스트 → iOS**: 호스트에서 텍스트를 복사하면 iOS 화면 하단에 Toast 알림이 나타나 복사된 내용의 미리보기가 표시됩니다. Toast를 탭하면 iOS 로컬 클립보드에 텍스트가 기록됩니다.
- **iOS → 호스트**: iOS 키보드에서 입력한 후 길게 눌러 붙여넣기하면 입력 내용이 호스트의 포커스 필드로 직접 전송됩니다.

---

## 5. 표시 크기 조정

툴바의 「🔍 원본 크기 / 화면 맞춤」 버튼으로 두 가지 표시 모드를 전환할 수 있습니다:

| 모드 | 설명 |
|---|---|
| **화면 맞춤** | 호스트 화면을 iOS 전체 화면에 맞게 스케일링 |
| **원본 크기** | 1:1 픽셀 표시, 패닝 지원, 세밀한 작업에 적합 |

---

## 6. 적응형 비트레이트（ABR）

2syn에는 자동 화질 조정 기능이 내장되어 있으며, 500밀리초마다 네트워크 RTT와 패킷 손실률을 감지합니다:

| 네트워크 상태 | 자동 조정 |
|---|---|
| 좋음（RTT < 80ms, 손실 < 1%）| 고화질, 높은 프레임레이트 |
| 보통 | 중간 화질 |
| 나쁨（RTT > 200ms 또는 손실 > 5%）| 프레임레이트와 비트레이트 감소, 연결 안정성 유지 |

오른쪽 상단의 점 표시기（녹색 / 노란색 / 빨간색）가 실시간으로 연결 품질을 반영합니다.

---

## 7. 프라이버시 블랙 스크린 모드

**호스트** 2syn 인터페이스에서 「프라이버시 모드」를 체크하면 호스트 화면이 검게 되어 주변 사람들이 작업 화면을 볼 수 없게 됩니다. 원격 제어는 계속 정상적으로 작동합니다.

---

## 8. 파일 전송

연결이 안정적일 때（P2P 직접 연결）, iOS 측 또는 호스트 측에서 2syn 창으로 파일을 드래그하면 전송이 시작됩니다. 모든 전송은 WebRTC Data Channel을 통해 엔드투엔드 암호화됩니다.

---

## 9. 연결 해제 및 재연결

툴바의 ⚙️ → 「🚪 로그아웃」을 탭하여 세션을 종료합니다. 연결이 예기치 않게 끊어진 경우 앱에 프롬프트가 표시되며 ID를 다시 입력하면 재연결할 수 있습니다.

---

## 10. 자주 묻는 질문

**Q: 연결이 「연결 중...」에 멈춰 설정되지 않는 경우？**
A: 양쪽 모두 엄격한 기업 방화벽（Symmetric NAT）뒤에 있으면 NAT 통과가 실패할 수 있습니다. 한쪽을 4G/5G 모바일 네트워크로 전환한 후 다시 시도해 보세요.

**Q: iOS에서 소리가 없는 경우？**
A: 화면의 「🔇 탭하여 오디오 활성화」 버튼을 탭하세요. iOS는 오디오 재생 잠금 해제를 위해 사용자 제스처가 필요합니다.

**Q: 화면이 흐리거나 끊기는 경우？**
A: 적응형 비트레이트（ABR）가 네트워크 상태에 따라 화질을 자동으로 조정합니다. 네트워크가 나쁠 때는 원활함을 유지하기 위해 화질이 낮아지고, 네트워크가 개선되면 자동으로 복구됩니다.

**Q: Apple Pencil의 필압 효과가 없는 경우？**
A: iPad의 2syn이 최신 버전인지, 호스트의 2syn도 v3.5.11 이상으로 업데이트되었는지 확인하세요.

**Q: QR 코드 스캔 후 연결되지 않는 경우？**
A: QR 코드에는 ID만 포함되어 있습니다. 호스트의 2syn이 실행 중이고 동일한 ID가 표시되는지 확인하세요.
